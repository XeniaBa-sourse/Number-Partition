Данное приложение работает в двух режимах:
    1) разбиение одного числа
    2) самоассоциированные разбиения
РАЗБИЕНИЯ ОДНОГО ЧИСЛА
Необходимо ввести натуральное число от 1 до 64 и выбрать с каким показателем работать (ранг или частное), нажать на кнопку "Получить результаты"
САМОАССОЦИИРОВАННЫЕ РАЗБИЕНИЯ
Неободимо ввести два натуральных числа (диапазон) и нажать на кнопку "Получить результаты"

В меню есть возможность сохранения данных в файле с расширением excel

Ограничение до числа 64 явялется условным, т.к. вычисление разбиений и частных весьма трудоемки.

app.py - реализация оконного интерфейса и обработка кнопок
data.py - представление данных в виде PandasDataFrame
Partitions - реализация класса для вычислений
for_plot - отрисовка графиков
factorization - представление частного в виде произведения простых чисел, разложенных по степеням
